Ra

Ruby Applets

Run Ruby commands independent of you operating systems execution system.

      $ gem install ra

List all available applets:

      $ ra help

Create a method, add it to a .rb script. For personal use, place the script in $HOME/.config/ra/. Applet ready.

To distribute you applet, save the script to your project's lib/plugins/ra folder. Distribution complete.


Tip: Don't use Dir.chdir, simply use cd instead.
